# Reyansh Turns Ten 🎉

A little birthday tribute website celebrating **Reyansh's 10th birthday**
(born 23 June 2016 — turning 10 on 23 June 2026, celebrated on 20 June).

It walks through his journey **year by year**, with photos collected from the
family's "Apna Group" chat over the past decade.

## How to view it

Just open **`index.html`** in any web browser — that's it. Everything is
self-contained (one HTML file plus the `images/` folder), so it works offline
and can be hosted anywhere (e.g. GitHub Pages).

## What's inside

- **`index.html`** — the whole webpage (HTML + CSS + a little JavaScript for the
  balloons, scroll animations, and tap-to-enlarge photos).
- **`images/`** — web-optimised photos, named by year (`y2017_a.jpg`, etc.).

## A note on the photos

The source was a media-only export of the group chat (no text messages), so the
photos were chosen by date — clustered around Reyansh's birthday each June, so
the **birthday boy in each set is Reyansh himself**. Coverage runs from his
**1st birthday (2017) to his 7th (2023)**; the export didn't include media for
2024–2026, so the most recent years aren't shown yet.

### Swapping or adding photos

1. Drop a new image into `images/`.
2. In `index.html`, find the matching `<img src="images/...">` line in that
   year's card and point it at your file.

That's all — no build step or tools required.

Made with love for Reyansh's 10th birthday. 🎂❤️🎈
